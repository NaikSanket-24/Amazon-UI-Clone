This project is a front-end clone of the Amazon homepage created for learning and practicing web development. The goal of this project was to understand how modern e-commerce websites structure their layouts and design user interfaces.

Technologies Used

- HTML
- CSS

Features

- Amazon-style navigation bar
- Product sections and categories
- Banner and promotional images
- Responsive layout structure

Project Structure

- index.html – Main webpage structure
- style.css – Styling and layout design
- images/ – Contains all images used in the project

Purpose

This project was built as part of my learning journey in web development to practice HTML and CSS and understand page layout, styling, and UI design.
